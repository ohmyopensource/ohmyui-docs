# OhMyUI! Docs

This repository contains the **official documentation for OhMyUI!**, the open source design system by **OhMyOpenSource!** .

> **Note:** This repository does not contain source code or components. For the actual library, see the OhMyUI! repository.

## What's inside

- **Getting started** - installation, setup and first component
- **Tokens** - colors, spacing, typography reference
- **Components** - usage guide for every component
- **Guides** - step by step tutorials for contributors
- **Technologies** - deep dives into Style Dictionary, Storybook, Nx and VitePress
